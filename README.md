# Аутентификация на React + Redux Toolkit + Firebase

Реализовал аутентификацию с помощью Firebase. Вход по email и паролю

Стэк: 
+ React
+ Redux toolkit
+ React hook form
+ Firebase 
+ Bootstrap
+ React-bootstrap 
+ React-router

Для запуска необходимо прописать npm install, далее npm start
